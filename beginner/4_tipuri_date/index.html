Tipuri de Date

Clasificarea tipului în JavaScript

Introducere

JavaScript are două tipuri de variabile: tipuri primitive și tipuri de referință. De fiecare dată când este creată o variabilă, o anumită dimensiune de memorie este rezervată. În plus, atunci când o variabilă este copiată, valoarea acesteia din memorie este, de asemenea, copiată. Când treceți o variabilă unei funcții, creați și o copie a variabilei

Tipuri primitive

JavaScript are 6 tipuri de date primitive: string, number, undefined, null, boolean și symbol. Aceste date pot stoca o singură valoare la un moment dat, deci sunt imutabile. Puteți modifica valoarea unei variabile care stochează o primitivă numai prin alocarea unei noi valori.

Boolean

Tipul boolean reprezintă o entitate logică care poate avea două valori: true sau false. Când definiți o variabilă, dacă valoarea este omisă sau este 0, -0, null, NaN, undefined sau un șir gol (""), variabila are valoarea false. În restul cazurilor va avea o valoare de true.

Referință 
// Boolean
let boolValue = true;

Null

Tipul null are exact o valoare: null. Reprezintă inexistența sau invaliditatea unei adrese de memorie sau a unui obiect. Este folosit în mod normal în variabilele de referință, deși în JavaScript utilizarea sa principală este primitivă.

Referință 
// Null
let nullValue = null;

Undefined

O variabilă căreia nu i s-a atribuit o valoare are valoarea undefined. Nu trebuie confundat cu null.

Referință 
// Undefined
let undefinedVal = undefined;

Number

ECMAScript are două tipuri de numere: number y bigint. Tipul de număr are o precizie de 64 de biți, care variază între -(2^53 − 1) și 2^53 − 1. În plus, valorile reprezentate cu coala flotantă au trei valori simbolice: +Infinity, -Infinity sau NaN (Nu este Număr).

Tipul BigInt este o primitivă numerică care poate reprezenta numere întregi cu precizie arbitrară. Prin urmare, ele pot reprezenta numere foarte mari pe care altfel numbernu poate.

Referință Referință Big Int 
// Number
let numberValue = 10;
let numBigIntVal = BigInt(9007199254740991);

String

String este folosit pentru a reprezenta informații textuale. Este un set de "elemente" pe 16 biți fără simbol. Fiecare element din tipul String ocupă o poziție. Primul element este 0 și nu 1 din interiorul indexului.

Spre deosebire de alte limbaje precum Python sau C, șirurile din JavaScript sunt imutabile. Aceasta înseamnă că odată create nu pot fi modificate. Totuși, alte șiruri pot fi create și atribuite aceleiași variabile cu operațiuni precum `String.substr()``

Referință 
//String
let stringVal = "Hello";
console.log(stringVal);

Symbol

Un simbol este un tip primitiv care poate fi folosit ca cheie pentru proprietatea unui obiect. În unele limbaje de programare se numesc atomi.

Referință 
// Symbol
let symVal = Symbol();

Tipuri după referință

Pe de altă parte, există un tip compus sau obiect. Aceste obiecte stochează o referință la un spațiu de memorie în care este stocată valoarea. Acest lucru permite modificarea valorii în timp și stocarea diferitelor valori în același timp.

Obiecte

Un obiect este o valoare din memorie referită de un identificator. În JavaScript, obiectele sunt cunoscute ca o colecție de proprietăți. JavaScript este un limbaj orientat spre prototip, în care obiectele stochează proprietăți, care pot fi adăugate sau eliminate. Aceste proprietăți pot fi valori de orice tip, inclusiv alte obiecte, permițând construirea unor structuri complexe de date.

Referință 
// Object
let emptyObject = {};
let filledObject = {"name": "Lucian", "surname": "Popa"};

Arrays(Matrice)

Sunt arrays obiecte în care există o relație între proprietățile cheii întregi și lungimea proprietății. În plus, moștenește de la obiectul Array.prototype care adaugă metode de manipulare a datelor cu index precum indexOf sau push.

În plus față de matrice, există și alte colecții cu chei care au fost adăugate în ES6, cum ar fi set și weakset care reprezintă seturi și Map și weakmap care reprezintă hărți de valori.

Referință 
// Array
let filledArray = ["one", "two", "three"];

Funcții

Mai târziu vom vedea funcțiile în profunzime, practic tipul de function permite stocarea funcțiilor în variabile, deoarece constructorul de funcții este derivat din constructorul Object.

Referință 
// Function
let funVale = function() {
    return "Hello World";
}
