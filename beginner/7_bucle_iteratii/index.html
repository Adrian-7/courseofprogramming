Buclă și iterație

Iterații în JavaScript

Există situații în care este necesar un bloc de instrucțiuni fie executat de un anumit număr de ori, fie pentru a itera între mai multe elemente, creați secvențe noi...

Pentru asta există instrucțiuni bucle. În JavaScript există mai multe mecanisme pentru a implementa repetarea codului, variînd unde este plasată evaluarea.

For

Bucla for este foarte asemănătoare cu buclele din C/C++ sau Java. O buclă for se repetă până când se ajunge la o evaluare false. Sintaxa principală este: 

for ([initialExpression]; [conditionExpression]; [incrementExpression])
  statement

Fluxul care urmează este următorul:

  1. Expresia inițială initialExpression este executată dacă există. Această expresie pornește un numărător, în plus această expresie poate declara variabile.
  2. Expresia conditionExpression este evaluată, dacă valoarea este true, blocul de cod este executat, dacă nu bucla se termină.
  3. Blocul de statement(instrucțiuni) este executat.
  4. Dacă există, se execută incrementedExpression
  5. Reveniți la pasul 2
for (let i = 0; i < 10; i++) {
    if (i%2 == 0) {
      console.log("Este număr par")
    } else {
        console.log("Nu este număr par")
    }
}

For In

Există un alt tip de buclă care iterează peste o proprietate enumerabilă sau cheile unui obiect. Această sintaxă este foarte utilă pentru trecerea în buclă prin iterabile. 

const object = { a: 1, b: 2, c: 3 };

for (const property in object) {
    console.log(`${property}: ${object[property]}`);
}

While

Folosind bucla while putem repeta un set de instrucțiuni atâta timp cât o condiție este îndeplinită. Sintaxa este foarte simplă: 

while (condition)
  statement

Când condiția este false, bucla este întreruptă și oprește executarea instrucțiunii.

Trebuie să fii foarte atent să nu creezi bucle infinite dacă creezi un contor 

let n = 0;
let x = 0;
while (n < 3) {
    n++;
    x += n;
    console.log(x);
}

// Buclă infinită aveți grijă deoarece n este egal cu 2
while (n > 2) {
    n++;
    x += n;
    console.log(x);
}

Do...While

Bucla do..while este similară cu cea anterioară, doar că mai întâi execută codul și apoi evaluează condiția. Este foarte util dacă vrem să ne asigurăm că expresia este executată cel puțin o dată. 

do
  statement
while (condition);

let i = 0;
do {
    i += 1;
    console.log(i);
} while (i < 5);

Switch

O instrucțiune switch poate înlocui mai multe verificări if. Oferă o modalitate mai descriptivă de a compara o valoare cu mai multe variante. 

switch (condition)
 case 1
  break
 case 2
  break
 default

Operația este următoarea, va evalua starea pe care o prezentăm în cadrul cuvântului cheie switch și va verifica toate cazurile în ordine descrescătoare până îl găsește pe cel corect sau îl execută implicit. 

let i = 3;
switch (a) {
  case 2:
    console.log( 'Prea mic' );
    break;
  case 3:
    console.log( 'Exact!' );
    break;
  case 4:
    console.log( 'Prea mare' );
    break;
  default:
    console.log( "Nu cunosc astfel de valori" );
}

De asemenea, putem grupa propoziții pentru a efectua aceeași acțiune în mai multe cazuri. 

switch(navigator.language) {
  case "en-US":
  case "en-GB":
  case "en-AU":
    console.log('Limba engleză');
    break;
  case "es-ES":
  case "es-AR":
  case "es-CL":
  case "es-CO":
  case "es-CR":
  case "es-DO":
    console.log('Limba spaniolă');
  default:
    console.log('Nicio limbă identificată')
}
