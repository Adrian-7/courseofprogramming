Operații condiționate în JavaScript

La cel mai elementar nivel, un program nu este altceva decât o execuție secvențială a instrucțiunilor. Această execuție are un flux predefinit pe care dezvoltatorul va dori să îl controleze cu diverse tehnici. Există momente când trebuie executate diferite acțiuni în funcție de condiții diferite. Pentru aceasta avem decizii condiționale care să execute deliberate instrucțiuni dependente de o condiție. Această condiție se reduce la un element boolean care este să spună true sau false.

JavaScript are două tipuri de sintaxă:

Cuvântul rezervat 'If'

Cuvântul cheie if(...) evaluează o condiție inclusă în paranteze, iar dacă rezultatul este true, execută o bucată de cod. 

let counter = 5
if(counter == 5) {
    console.log("Am ajuns la 5")
}

Dacă ne întoarcem la definiția booleanului, putem vedea că există anumite tipuri care returnează false atunci când sunt evaluate, în timp ce celelalte vor returna true.

If ... else

Putem adăuga una sau mai multe clauze în blocul nostru condiționat pentru a executa condiția care este adevărată, aceste execuții vor fi evaluate în cascadă și vor executa toate condițiile. 

if(counter == 6) {
    print("Am ajuns la 6");
    console.log("Salut");
} else {
    console.log("Mai avem ceva");
}

Acesta este motivul pentru care putem cuibări mai multe condiții

counter += 1

if(counter == 1){
    console.log("Am ajuns la 1");
} else if(counter == 2){
    console.log("Am ajuns la 2");
} else if(counter == 3){
    console.log("Am ajuns la 3");
} else if(counter == 4){
    console.log("Am ajuns la 4");
} else if(counter == 5){
    console.log("Am ajuns la 5");
} else if(counter == 6){
    console.log("Am ajuns la 6");
} else {
    console.log("Avem mai mult decât atât");
}
