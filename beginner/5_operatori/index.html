Operatori

Operații în JavaScript

Înainte de a enumera diferiții operatori, există o terminologie pe care trebuie să o acoperim:

  • Un operand este elementul căruia i se aplică operatorii. De exemplu, într-o sumă 2 + 3, operatorul din stânga este 2, operatorul din dreapta este 3 și operandul este +

  • Un operator este unar dacă are un singur operator, cum ar fi negația -3 în timp ce este binar dacă are doi operanzi, 2 + 3.

  • Precedența operatorilor indică ordinea de operare. Dacă în aceeași linie de execuție găsim diferiți operatori și cuvinte rezervate, JavaScript le va executa în ordinea indicată în această listă.

Operatori aritmetici

Operatorii aritmetici modifică valoarea unei variabile. Acestea urmează ordinea de prioritate menționată mai sus. 

// Operatori aritmetici
let a = 30;
let b = 50;

// Adunarea
let add = a + b;
console.log(`add: ${add}`)

// Scăderea
let sub = a - b
console.log(`sub: ${sub}`)

// Înmulțirea
let mul = a * b
console.log(`mul: ${mul}`)

// Împărțirea
let div = a / b
console.log(`div: ${div}`)

// Modulul
let modd = b % a
console.log(`modd: ${modd}`)

// Exponentul
let exp = a**b
console.log(`exp: ${exp}`)

Operatori de comparare

Ei compară valorile din ambele părți ale tranzacției și decid relația dintre cele două. 

// Operatori de comparare
a = 30;
b = 50;

// Egalitate
let eq = a == b;
console.log(`eq: ${eq}`);

let eq2 = a == c;
console.log(`eq2: ${eq2}`);

// Egalitate + tip
let eq3 = a === c;
console.log(`eq3: ${eq3}`);

// Inegalitate
let ieq = a != b;
console.log(`ieq: ${ieq}`);

let ieq2 = a != c;
console.log(`ieq2: ${ieq2}`);

// Inegalitate + tip
let ieq = a !== b;
console.log(`ieq: ${ieq}`);

// Mai mare
let gt = a > b;
console.log(`gt: ${gt}`);

// Mai mic
let ls = a < b;
console.log(`ls: ${ls}`);

// Mai mare sau egal
let gteq = a >= b;
console.log(`gteq: ${gteq}`);

// Mai mic sau egal
let lseq = a <= b;
console.log(`lseq: ${lseq}`);

console.log(30 > 30);
console.log(30 >= 30);

Operatori de atribuire

Am văzut deja acești operatori de mai multe ori când am declarat variabile, aici vom vedea câteva dintre proprietățile lor. 

// Operatori de atribuire
a = 30;
b = 50;

// Atribuire (=)
let c = a + b;
console.log(c);

// Adăugare cu atribuire
c = 60;
c += a;
console.log(c);

// Scăderea cu atribuire
c = 60;
c -= a;
console.log(c);

// Înmulțire cu atribuire
c = 60;
c *= a;
console.log(c);

// Împărțirea cu atribuire
c = 60;
c /= a;
console.log(c);

Operatori logici

Acești operatori sunt utilizați în cadrul controalelor de flux pe care le vom vedea în secțiunea următoare. Toate returnează un tip logic (adevărat sau fals) atunci când operația este efectuată. 

// Operatori logici
a = 30;
b = 50;
c = [20, 30, 70, 80];
let at = true;
let bf = false;

// AND operator
console.log(at && bf);

// OR operator
console.log(at || bf);

// NOT operator
console.log(!at);

console.log("------------")

// Membership
console.log(a in c);
